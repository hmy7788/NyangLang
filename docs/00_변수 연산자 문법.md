# NyangLang 기본 문법
본 문서는 NyangLang의 `기본 문법`을 정의한다.
1. 기본 개념
2. 변수
3. 연산




## 1. 기본 개념
### 1.1 프로그램 구조
- 한 줄에 `여러 명령문`을 작성할 수 있다.
- 모든 명령문은 반드시 `야옹`으로 끝나야 한다.
- 모든 계산에 사용되는 숫자와 연산 결과는 `스택(stack)` 자료구조를 통해 관리된다.
- 모든 의미는 해당 명령문에 포함된 세 가지 요소로 결정된다.
  1. 냥의 개수 (ID)
  2. 냐의 존재
  3. 문자 (`.`, `,`, `~`, `!`, `?`) 와 종료자 (`야옹`)

그 외의 모든 문자(공백, 한글, 숫자, 영어, 기호)는 주석으로 간주하여 무시한다.  
단, `야옹`은 명령문 종료자로 인식된다.

### 1.2 ID 규칙
- 명령문에 등장하는 `냥`의 개수를 N이라 할 때, 해당 명령문의 ID는 N이다.
- ID는 다음과 같이 사용된다:
  - 변수 번호
  - 함수 번호
  - 배열 번호
  - 조건문/반복문의 제어 변수




## 2. 정수형
정수: 온점, 반점의 갯수로 나타낸다. 온점의 갯수만큼 1을 더하며, 반점의 갯수만큼 1을 뺀다.
예시:
```
...   # 3
,,,   # -3
.,,   # -1
```




## 3. 변수(Variable)와 상수
### 3.1 변수 선언
형식:
```
<냥 N개><.과 , 조합>야옹
```
의미:
- `N은 냥 개수`이며, `변수 번호`를 나타낸다.
- `.과 , 조합`은 변수의 `초기값`을 나타낸다.
- 초기값이 0인 경우에도 반드시 `.,` 또는 `.` 등 값을 명시해야 한다. (`냥야옹`만 쓰는 것은 허용하지 않는다.)

예시:
```
냥.,야옹          # 변수1 = 0
냥냥....야옹      # 변수2 = 4
냥냥냥.....야옹   # 변수3 = 5
냥냥냥냥,,,야옹   # 변수4 = -3
냥냥냥냥냥..,야옹 # 변수5 = 1
```




### 3.2 변수 값의 사용과 대입
형식:
```
1. <. 과 , 조합>~야옹
2. <냥 N개>~야옹
3. <냥 N개>~~야옹
```
의미:
1. `<. 과 , 조합>~야옹` : 정수형을 스택에 push 한다. (VALUE_PUSH)
2. `<냥 N개>~야옹` : 변수 N의 값을 스택에 push 한다. (VAR_PUSH)
3. `<냥 N개>~~야옹` : 스택의 top 값을 1개 pop 하여 변수 N에 대입한다. (VAR_ASSIGN)

예시:
```
[VALUE_PUSH]
...~야옹    # 스택에 3 push
,~야옹      # 스택에 -1 push

[VAR_PUSH]
냥냥....야옹    # 변수2 = 4
냥냥~야옹       # 스택에 4 push

[VAR_ASSIGN]
냥냥....야옹    # 변수2 = 4
냥냥냥.,야옹    # 변수3 = 0
냥냥~야옹       # 스택에 4 push
...~야옹        # 스택에 3 push
냐냐~야옹       # 덧셈: 스택에서 2개 pop(4, 3) → 7 push
냥냥냥~~야옹    # 스택 top(7) pop → 변수3 = 7
```




## 4. 연산(Operation)
`모든 연산은 스택 기반으로 수행`된다.
절차:
1. 스택에서 두 개의 값을 pop
2. 지정된 연산을 수행
3. 연산 결과를 스택에 push

### 4.1 연산 규칙
형식:
```
<냐 N개>~야옹
```
|냐 개수|연산|
|---|---|
|2|덧셈|
|3|뺄셈|
|4|곱셈|
|5|나눗셈(정수)|

예시:
```
...~야옹    # 스택에 3 push
....~야옹   # 스택에 4 push
냐냐~야옹   # 덧셈: pop(3, 4) → 7 push

...~야옹    # 스택에 3 push
....~야옹   # 스택에 4 push
냐냐냐~야옹 # 뺄셈: pop(3, 4) → -1 push
```
