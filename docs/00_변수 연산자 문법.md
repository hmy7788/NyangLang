## 1. 기본 개념

### 프로그램 구조

- 한 줄에 **여러 명령문**을 작성할 수 있습니다.
- 모든 명령문은 반드시 **`야옹`** 으로 끝나야 합니다.
- 모든 계산은 **스택(stack)** 자료구조를 통해 관리됩니다.
- `야옹` 외의 모든 문자(공백, 한글, 숫자, 영어, 기호)는 **주석**으로 간주합니다.

### ID 규칙

명령문에 등장하는 `냥`의 개수 **N** 이 해당 명령문의 ID입니다.

| ID 용도 | 설명 |
|---------|------|
| 변수 번호 | `냥냥` → 변수2 |
| 배열 번호 | `냥냥냥` → 배열3 |
| 조건/반복 제어 변수 | 점프문의 조건 변수 |

### 명령문 구성 요소

명령문의 의미는 아래 세 가지 요소의 조합으로 결정됩니다.

| 요소 | 역할 |
|------|------|
| `냥`의 개수 | 변수/배열 ID |
| `냐`의 존재 | 연산 명령 여부 |
| 특수문자 (`.` `,` `~` `!` `?`) + `야옹` | 명령의 종류 |

## 2. 정수 리터럴

`.`(점)과 `,`(반점)의 조합으로 정수를 표현합니다.

| 규칙 | 설명 |
|------|------|
| `.` 1개 | +1 |
| `,` 1개 | -1 |

```
...    # 3  (점 3개)
,,,    # -3 (반점 3개)
.,,    # -1 (1 - 2 = -1)
.,     # 0  (1 - 1 = 0)
```

## 3. 변수

### 3-1. 변수 선언

```
냥N <값>야옹
```

- `N` = `냥`의 개수 (변수 번호)
- `<값>` = `.`과 `,`의 조합으로 초기값 표현
- 초기값이 0이더라도 반드시 값을 명시해야 합니다. (`냥야옹`만 쓰는 것은 허용하지 않습니다.)

**예시**

```
냥.,야옹          # 변수1 = 0
냥냥....야옹      # 변수2 = 4
냥냥냥.....야옹   # 변수3 = 5
냥냥냥냥,,,야옹   # 변수4 = -3
```

---

### 3-2. 값 사용 및 대입

| 구문 | 종류 | 의미 |
|------|------|------|
| `<값>~야옹` | VALUE_PUSH | 정수 리터럴을 스택에 push |
| `냥N~야옹` | VAR_PUSH | 변수N의 값을 스택에 push |
| `냥N~~야옹` | VAR_ASSIGN | 스택 top을 pop하여 변수N에 대입 |

**예시**

```
# VALUE_PUSH
...~야옹    # 스택에 3 push
,~야옹      # 스택에 -1 push

# VAR_PUSH
냥냥....야옹    # 변수2 = 4
냥냥~야옹       # 스택에 4 push

# VAR_ASSIGN
냥냥....야옹    # 변수2 = 4
...~야옹        # 스택에 3 push
냐냐~야옹       # 덧셈: 4 + 3 = 7
냥냥냥~~야옹    # 변수3 = 7 (스택 top pop)
```

## 4. 연산

모든 연산은 **스택 기반**으로 수행됩니다.

> 스택에서 값 두 개를 pop → 연산 수행 → 결과를 스택에 push

### 연산 규칙

```
냐N개~야옹
```

| `냐` 개수 | 연산 | 예시 |
|-----------|------|------|
| 2 | 덧셈 | `냐냐~야옹` |
| 3 | 뺄셈 | `냐냐냐~야옹` |
| 4 | 곱셈 | `냐냐냐냐~야옹` |
| 5 | 나눗셈 (정수) | `냐냐냐냐냐~야옹` |
| 6 | 나머지 | `냐냐냐냐냐냐~야옹` |

> **뺄셈 주의**: 스택에서 먼저 push된 값이 피감수, 나중에 push된 값이 감수입니다.  
> 즉, `A push → B push → 뺄셈` = `A - B`

**예시**

```
# 덧셈: 3 + 4 = 7
...~야옹    # 3 push
....~야옹   # 4 push
냐냐~야옹   # 7 push

# 뺄셈: 10 - 3 = 7
..........~야옹    # 10 push
...~야옹           # 3 push
냐냐냐~야옹        # 7 push

# 곱셈: 3 × 4 = 12
...~야옹           # 3 push
....~야옹          # 4 push
냐냐냐냐~야옹      # 12 push

# 나머지: 10 % 3 = 1
..........~야옹         # 10 push
...~야옹                # 3 push
냐냐냐냐냐냐~야옹       # 1 push
```
