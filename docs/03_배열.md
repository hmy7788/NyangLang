# NyangLang 배열
본 문서는 NyangLang의 `배열`를 정의한다.
배열 관련 기능은 아래 3가지로 나뉜다.
- `배열 생성`
- `배열 값 쓰기`
- `배열 값 읽기`


배열의 이름과 변수의 이름이 같아도 무관하며, 동일한 이름을 사용해도 별도로 취급된다. <br>
모든 배열의 인덱스는 0부터 시작한다.



## 배열 생성
- 배열의 생성은 
- 배열명은 변수명과 상관없다. 즉, 이름이 중복되어도 된다.
- 배열을 생성할때 초기값은 모두 0으로 초기화한다.

형식:
```
<배열명>!<정수형/변수형>
```

설명:
1. `<배열명>`이라는 이름의 배열을 생성한다.
2. `<정수형>` 또는 `변수형`이 나타내는 숫자를 배열의 길이로 사용한다.
3. 생성된 배열의 모든 요소는 `0`으로 초기화한다.

예제:
```
냥!.....    # 길이가 5인 배열 '냥' 생성
냥냥!...    # 길이가 3인 배열 '냥냥' 생성
냥냥냥!냥   # 변수 '냥'의 값만큼의 길이를 가진 배열 '냥냥냥' 선언
```


## 배열 값 쓰기
형식:
```
<배열명>!<정수형1/변수형1>!<정수형2/변수형2>
```

설명:
1. `<정수형1>` 또는 `<변수형1>`을 사용해 `인덱스`를 지정할 수 있다.
2. `<정수형2>` 또는 `<변수형2>`을 사용해 `값`을 지정할 수 있다.
3. `<배열명>`의 `인덱스` 위치에 `값`을 저장한다.

예제:
```
냥!.....!......         # 배열 '냥'의 5번 인덱스에 6 저장
냥냥!.,!....            # 배열 '냥냥'의 0번 인덱스에 4 저장
냥냥냥!냥!.,            # 배열 '냥냥냥'의 (변수 '냥'의 값) 위치에 0을 저장
냥냥냥냥!냥냥!냥냥냥    # 배열 '냥냥냥냥'의 (변수 '냥냥'의 값) 위치에 (변수 '냥냥냥')의 값 저장
```


## 배열 값 읽기
형식:
```
<배열명>!?<정수형1/변수형1>!?<~/변수형2>
```


설명:
배열 값을 읽는 방식은 두 가지이다.
1. 스택에 push
    ```
    <배열명>!?<정수형1/변수형1>!?~
    ```
    배열의 정수형 또는 변수형 인덱스 위치의 값을 스택에 push
2. 변수에 대입
    ```
    <배열명>!?<정수형1/변수형1>!?<변수형2>
    ```
    배열의 정수형 또는 변수형 인덱스 위치에 값을 변수형2에 대입



예제:
```
냥!?.....!?~            # 배열 '냥'의 5번 인덱스 값을 스택에 push
냥냥!?.,!?냥냥          # 배열 '냥냥'의 0번 인덱스 값을 변수 '냥냥'에 대입
냥냥냥!?냥!?~           # 배열 '냥냥냥'의 변수 '냥'의 값 인덱스의 값을 스택에 push
냥냥냥냥!?냥냥!?냥냥냥  # 배열 '냥냥냥냥'의 변수 '냥냥'의 값 인덱스의 값을 변수 '냥냥냥'에 대입
```
