# NyangLang 배열
본 문서는 NyangLang의 `배열`을 정의한다.
배열 관련 기능은 아래 3가지로 나뉜다.
- `배열 생성`
- `배열 값 쓰기`
- `배열 값 읽기`


배열의 이름과 변수의 이름이 같아도 무관하며, 동일한 이름을 사용해도 별도로 취급된다. <br>
모든 배열의 인덱스는 0부터 시작하며, 모든 요소는 0으로 초기화된다. <br>
배열의 ID는 명령문에 사용된 `냥`의 개수(N)로 결정된다.




## 1. 배열 생성 (ARRAY_DECL)

형식:
```
<냥 N개>!<정수형/변수형>야옹
```

의미:
- `냥 N개` : 배열의 ID(번호)를 나타낸다.
- `!<정수형>` : 정수형 값을 배열의 `길이`로 사용한다.
- `!<변수형>` : 해당 변수의 값을 배열의 `길이`로 사용한다.

예시:
```
냥!.....야옹       # 배열1 생성, 길이 5
냥냥!...야옹       # 배열2 생성, 길이 3
냥냥냥!냥야옹      # 배열3 생성, 길이 = 변수1의 값
```




## 2. 배열 값 쓰기 (ARRAY_WRITE)

형식:
```
<냥 N개>!<정수형1/변수형1>!<정수형2/변수형2>야옹
```

의미:
- `<정수형1>` 또는 `<변수형1>` : 쓸 위치의 `인덱스`를 지정한다.
- `<정수형2>` 또는 `<변수형2>` : 배열에 저장할 `값`을 지정한다.
- 배열 N의 `인덱스` 위치에 `값`을 저장한다.

| 인덱스 타입 | 값 타입 | 예시 |
|-------------|---------|------|
| 정수형      | 정수형  | `냥!.....!......야옹` |
| 정수형      | 변수형  | `냥!.!냥냥야옹` |
| 변수형      | 정수형  | `냥!냥냥!...야옹` |
| 변수형      | 변수형  | `냥!냥냥!냥냥냥야옹` |

예시:
```
냥!.....!......야옹        # 배열1의 5번 인덱스에 6 저장
냥냥!.,!....야옹           # 배열2의 0번 인덱스에 4 저장
냥냥냥!냥!.,야옹           # 배열3의 (변수1의 값)번 인덱스에 0 저장
냥냥냥냥!냥냥!냥냥냥야옹   # 배열4의 (변수2의 값)번 인덱스에 변수3의 값 저장
```




## 3. 배열 값 읽기 (ARRAY_READ)

형식:
```
<냥 N개>!?<정수형1/변수형1>!?<~/변수형2>야옹
```

의미:
- `<정수형1>` 또는 `<변수형1>` : 읽을 위치의 `인덱스`를 지정한다.
- `!?~` : 읽은 값을 `스택에 push`한다.
- `!?<변수형2>` : 읽은 값을 `변수에 대입`한다.

| 인덱스 타입 | 목적지   | 예시 |
|-------------|----------|------|
| 정수형      | 스택     | `냥!?.....!?~야옹` |
| 정수형      | 변수형   | `냥냥!?.,!?냥냥야옹` |
| 변수형      | 스택     | `냥냥냥!?냥!?~야옹` |
| 변수형      | 변수형   | `냥냥냥냥!?냥냥!?냥냥냥야옹` |

예시:
```
냥!?.....!?~야옹             # 배열1의 5번 인덱스 값을 스택에 push
냥냥!?.,!?냥냥야옹           # 배열2의 0번 인덱스 값을 변수2에 대입
냥냥냥!?냥!?~야옹            # 배열3의 (변수1의 값)번 인덱스 값을 스택에 push
냥냥냥냥!?냥냥!?냥냥냥야옹   # 배열4의 (변수2의 값)번 인덱스 값을 변수3에 대입
```
