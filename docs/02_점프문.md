## 1. 개요

점프문은 NyangLang의 **조건 분기와 반복문** 을 담당합니다.  
조건값이 0이 아니면 지정한 라인으로 이동하고, 0이면 다음 줄을 실행합니다.

---

## 2. 문법

```
<조건>?<점프 라인>야옹
```

- `?` 기준으로 왼쪽이 **조건**, 오른쪽이 **점프할 라인 번호** 입니다.
- 조건과 점프 라인 각각에 **정수형** 또는 **변수형** 을 사용할 수 있습니다.

### 4가지 조합

| 종류 | 형식 | 의미 |
|------|------|------|
| 정수 → 정수 라인 | `<정수>?<정수>야옹` | 정수값 ≠ 0이면 정수 라인으로 점프 |
| 정수 → 변수 라인 | `<정수>?냥N야옹` | 정수값 ≠ 0이면 변수N번 라인으로 점프 |
| 변수 → 정수 라인 | `냥N?<정수>야옹` | 변수N ≠ 0이면 정수 라인으로 점프 |
| 변수 → 변수 라인 | `냥N?냥M야옹` | 변수N ≠ 0이면 변수M번 라인으로 점프 |

> **라인 번호**는 파일의 첫 번째 줄이 1번입니다.

---

## 3. 조건 규칙

| 조건값 | 동작 |
|--------|------|
| **0이 아닌 값** | 지정한 라인으로 **점프** |
| **0** | 점프 없이 **다음 줄 순차 실행** |

### 무조건 점프

조건을 항상 0이 아닌 고정값으로 설정하면 무조건 점프가 됩니다.

```
.?...야옹    # 항상 점프 (1 ≠ 0이므로 3번 라인으로)
```

---

## 4. 예제

### 예제 1: 1부터 5까지 출력 (반복문)

```
(1)  냥.야옹           # 변수1 = 1
(2)  냥냥.....야옹     # 변수2 = 5 (카운터)
(3)  냥!?야옹          # 변수1 출력
(4)  냥~야옹           # 변수1 push
(5)  .~야옹            # 1 push
(6)  냐냐~야옹         # 덧셈: 변수1 + 1
(7)  냥~~야옹          # 변수1 = 결과
(8)  냥냥~야옹         # 변수2 push
(9)  ,~야옹            # -1 push
(10) 냐냐~야옹         # 덧셈: 변수2 - 1
(11) 냥냥~~야옹        # 변수2 = 결과
(12) 냥냥?...야옹      # 변수2 ≠ 0이면 3번 라인으로 점프 → 반복
```

**출력**: `1 2 3 4 5`

---

### 예제 2: 짝수/홀수 판별

입력받은 수가 짝수면 `1`, 홀수면 `0` 출력합니다.

```
(1)  냥??야옹                  # 변수1 ← 입력
(2)  냥?.....야옹              # 변수1 = 0이면 line 5, 아니면 line 3
(3)  .!?야옹                   # 1 출력 (짝수)
(4)  .?.................야옹   # line 17로 무조건 점프 (종료)
(5)  냥~야옹                   # 변수1 push
(6)  .~야옹                    # 1 push
(7)  냐냐냐~야옹               # 뺄셈: 변수1 - 1
(8)  냥~~야옹                  # 변수1 = 결과
(9)  냥?............야옹       # 변수1 ≠ 0이면 line 12
(10) ,.!?야옹                  # 0 출력 (홀수)
(11) .?.................야옹   # line 17로 무조건 점프 (종료)
(12) 냥~야옹
(13) ..~야옹
(14) 냐냐냐~야옹
(15) 냥~~야옹
(16) .?..야옹                  # line 2로 무조건 점프
(17)                           # 프로그램 종료
```

---

### 예제 3: 변수 라인 점프 (동적 점프)

점프할 라인 번호를 **변수에 저장**해 동적으로 분기할 수 있습니다.

```
(1) 냥.야옹        # 변수1 = 1 (조건)
(2) 냥냥....야옹   # 변수2 = 4 (점프 대상 라인)
(3) 냥?냥냥야옹    # 변수1 ≠ 0이면 변수2(=4)번 라인으로 점프
(4) ...!?야옹      # 3 출력 (점프 목적지)
```
