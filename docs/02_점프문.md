# NyangLang 점프문
본 문서는 NyangLang의 `점프문`을 정의한다.
1. <숫자형>?<숫자형>
2. <숫자형>?<변수형>
3. <변수형>?<숫자형>
4. <변수형>?<변수형>
- `?`을 기준으로 왼쪽을 `조건`, 오른쪽을 `점프할 라인`이라 정의한다.
- 변수형이 등장할 경우, 해당 변수는 현재 저장된 값으로 평가하여 사용한다.


## 조건
- `조건의 값이 0이 아니면 점프할 라인으로 점프`한다.
- `조건의 값이 0이면 그대로 순차 실행`한다.


예시1:
```
[1~5 정수를 출력하는 코드]
(1) 냥.             # 변수1 = 1
(2) 냥냥.....       # 변수2 = 5
(3) 냥!             # 변수1 출력
(4) 냥~             # 변수1 push
(5) .~              # 1 push
(6) 냐냐~           # 덧셈 연산
(7) 냥~             # 변수1 = 2
(8) 냥냥~           # 변수2 push
(9) ,~              # -1 push
(10) 냐냐~          # 덧셈 연산
(11) 냥냥~          # 변수2 = 4 . . .
(12) 냥냥?...       # 냥냥 != 0이면 3번 라인 점프, 냥냥 == 0이면 순차진행
```

예시2:
```
[사용자 입력을 받아 짝수인지 홀수인지 판별하는 코드]
[짝수면 1 출력, 홀수면 0 출력]
(1) 냥?
(2) 냥?.....                # 냥 값이 0이면 line 3 실행, 아니면 line 5 점프
(3) .!                      # 1 출력
(4) .?.................     # line 17로 점프 (무조건 점프)
(5) 냥~                     # 냥 값 push
(6) .~                      # 1 push
(7) 냐냐냐~                 # 뺄셈 연산
(8) 냥냥~                   # 
(9) 냥냥?............       # 냥냥 값이 0이면 line 10 실행, 아니면 12 line 점프
(10) ,.!                    # 0 출력
(11) .?.................    # line 17로 점프 (무조건 점프)
(12) 냥~                    # 
(13) ..~                    # 
(14) 냐냐냐~                # 
(15) 냥~                    # 
(16) .?..                   # line 2로 점프 (무조건 점프)
(17)                        # 프로그램 종료
```
