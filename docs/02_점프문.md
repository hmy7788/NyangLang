# NyangLang 점프문
본 문서는 NyangLang의 `점프문`을 정의한다.

형식:
```
<조건><QUESTION_1><점프라인>야옹
```
- `?`을 기준으로 왼쪽을 `조건`, 오른쪽을 `점프할 라인`이라 정의한다.
- `조건`과 `점프라인` 각각에 정수형(`INT`) 또는 변수형(`냥 N개`) 사용 가능하다.
- 변수형이 등장할 경우, 해당 변수에 저장된 값으로 평가한다.

지원하는 4가지 조합:
| 종류 | 형식 | 의미 |
|------|------|------|
| int?int | `INT?INT야옹` | 정수값이 0이 아니면 정수 라인으로 점프 |
| int?nyang | `INT?냥N야옹` | 정수값이 0이 아니면 변수N번 라인으로 점프 |
| nyang?int | `냥N?INT야옹` | 변수N이 0이 아니면 정수 라인으로 점프 |
| nyang?nyang | `냥N?냥M야옹` | 변수N이 0이 아니면 변수M번 라인으로 점프 |


## 조건
- `조건의 값이 0이 아니면 점프할 라인으로 점프`한다.
- `조건의 값이 0이면 그대로 순차 실행`한다.
- 무조건 점프는 조건을 항상 0이 아닌 값으로 고정하면 된다. (예: `.?...야옹`)


예시1:
```
[1~5 정수를 출력하는 코드]
(1)  냥.야옹           # 변수1 = 1
(2)  냥냥.....야옹     # 변수2 = 5
(3)  냥!?야옹          # 변수1 출력
(4)  냥~야옹           # 변수1 push
(5)  .~야옹            # 1 push
(6)  냐냐~야옹         # 덧셈 연산
(7)  냥~~야옹          # 변수1 = 연산결과
(8)  냥냥~야옹         # 변수2 push
(9)  ,~야옹            # -1 push
(10) 냐냐~야옹         # 덧셈 연산
(11) 냥냥~~야옹        # 변수2 = 연산결과
(12) 냥냥?...야옹      # 변수2 != 0이면 3번 라인 점프, == 0이면 순차진행
```

예시2:
```
[사용자 입력을 받아 짝수인지 홀수인지 판별하는 코드]
[짝수면 1 출력, 홀수면 0 출력]
(1)  냥??야옹                  # 변수1 ← 입력
(2)  냥?.....야옹              # 변수1 값이 0이면 line 3 실행, 아니면 line 5 점프
(3)  .!?야옹                   # 1 출력
(4)  .?.................야옹   # line 17로 점프 (무조건 점프)
(5)  냥~야옹                   # 변수1 push
(6)  .~야옹                    # 1 push
(7)  냐냐냐~야옹               # 뺄셈 연산
(8)  냥~~야옹                  # 변수1 = 변수1 - 1
(9)  냥?............야옹       # 변수1 != 0이면 line 12 점프, == 0이면 line 10 실행
(10) ,.!?야옹                  # 0 출력
(11) .?.................야옹   # line 17로 점프 (무조건 점프)
(12) 냥~야옹
(13) ..~야옹
(14) 냐냐냐~야옹
(15) 냥~~야옹
(16) .?..야옹                  # line 2로 점프 (무조건 점프)
(17)                           # 프로그램 종료
```

예시3 (변수 라인 점프):
```
[nyang?nyang: 변수에 저장된 라인 번호로 점프]
(1) 냥.야옹        # 변수1 = 1 (조건)
(2) 냥냥....야옹   # 변수2 = 4 (점프 대상 라인)
(3) 냥?냥냥야옹    # 변수1 != 0이면 변수2(=4)번 라인으로 점프
(4) ...!?야옹      # 3 출력 (점프 목적지)
```
