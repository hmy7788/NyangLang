## 1. 입력

```
냥N??야옹
```

정수를 입력받아 **변수N** 에 저장합니다.

**예시**

```
냥??야옹    # 변수1 ← 정수 입력
냥냥??야옹  # 변수2 ← 정수 입력
```

---

## 2. 출력

### 2-1. 숫자 출력

출력은 **종류 × 형식 × 모드** 의 조합으로 결정됩니다.

#### 출력 종류

| 종류 | 표현 | 설명 |
|------|------|------|
| 변수형 | `냥N` | 변수N의 값을 출력 |
| 정수형 | `<값>` | 정수 리터럴을 출력 |

#### 출력 형식

| 형식 | 기호 | 설명 |
|------|------|------|
| 십진수 | `!` | 정수 값 그대로 출력 |
| ASCII | `!!` | 해당 코드의 문자 출력 |

#### 출력 모드

| 모드 | 기호 | 설명 |
|------|------|------|
| 줄바꿈 | `?` | 출력 후 `\n` |
| 인라인 | `??` | 출력 후 아무것도 없음 |
| 공백 | `???` | 출력 후 공백 1칸 |

#### 조합 규칙

세 축을 곱하면 `종류(2) × 형식(2) × 모드(3) = 12가지` 출력이 가능합니다.

```
<종류><형식><모드>야옹
```

**예시** — 변수2 = 68 기준 (ASCII 68 = `'D'`)

| 구문 | 형식 · 모드 | 출력 |
|------|-------------|------|
| `냥냥!?야옹` | 십진수 · 줄바꿈 | `68\n` |
| `냥냥!??야옹` | 십진수 · 인라인 | `68` |
| `냥냥!???야옹` | 십진수 · 공백 | `68 ` |
| `냥냥!!?야옹` | ASCII · 줄바꿈 | `D\n` |
| `냥냥!!??야옹` | ASCII · 인라인 | `D` |
| `냥냥!!???야옹` | ASCII · 공백 | `D ` |

정수형도 동일하게 동작합니다. 예를 들어 `....!?야옹`은 정수 4를 십진수·줄바꿈으로 출력하여 `4\n`이 됩니다.

---

### 2-2. 디버깅 출력

프로그램 개발/디버깅 시 현재 내부 상태를 확인할 수 있습니다.

| 구문 | 출력 내용 |
|------|-----------|
| `냐!야옹` | 현재 스택 상태 |
| `냐!!야옹` | 현재 변수 테이블 전체 |
