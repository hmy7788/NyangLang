# NyangLang 사용자 입출력 문법
본 문서는 NyangLang의 `사용자 입출력 문법`을 정의한다.
1. 입력
2. 출력




## 1. 입력(input)
형식:
```
<냥 N개>?
```
의미:
- 사용자로부터 `정수 입력`을 받아 `변수 N에 저장`한다.

예시:
```
냥?     # 변수1 ← 입력
냥냥?   # 변수2 ← 입력
```




## 2. 출력(output)
- 일반적인 출력은 `출력 종류(정수형, 변수형)`와 `출력 형식(십진수, ASCII)`과 `출력 모드(newline, inline, space)`의 조합으로 나타낸다.
1. 출력 종류(kind)
  - `정수형`: 정수형을 출력한다.
  - `변수형`: 변수의 값을 출력한다.
  
2. 출력 형식(form)
  - `!`: 십진수로 출력한다.
  - `!!`: ASCII 문자로 출력한다.

3. 출력 모드(mode)
  - `?`: 출력을 하고 줄바꿈을 한다. (newline)
  - `??`: 출력을 하고 줄바꿈을 하지않는다. (inline)
  - `???`: 출력을 하고 공백 1칸을 출력한다. (space)

- 표현식은 다음과 같이 구성된다.
```
<정수형/변수형><!/!!><?/??/???>
``` 
총 12개의 출력 조합을 만들 수 있다.

조합: 아래의 모든 조합이 허용된다.
```
1. <정수형>!?
2. <정수형>!?? 
3. <정수형>!???
4. <정수형>!!?
5. <정수형>!!??
6. <정수형>!!???
7. <변수형>!?
8. <변수형>!?? 
9. <변수형>!???
10. <변수형>!!?
11. <변수형>!!??
12. <변수형>!!???
```

- 내장함수(디버깅) 출력
내장함수는 위 조합을 따르지 않는다.
  1. 현재 스택 출력: `냐!`
  2. 현재 변수 테이블 출력: `냐!!`
