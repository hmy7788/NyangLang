# NyangLang 사용자 입출력 문법
본 문서는 NyangLang의 `사용자 입출력 문법`을 정의한다.
1. 입력
2. 출력




## 1. 입력(input)
형식:
```
<냥 N개>?
```
의미:
- 사용자로부터 `정수 입력`을 받아 `변수 N에 저장`한다.

예시:
```
냥?     # 변수1 ← 입력
냥냥?   # 변수2 ← 입력
```




## 2. 출력(output)
형식:
```
1. <정수형>!
2. <냥 N개>!
3. <냥 N개>!!
```
의미:
1. `<정수형>!` <br>
정수를 `십진수 형태`로 출력해준다.
2. `<정수형>!!` <br>
정수형을 `ASCII 코드`로 해석하여 해당 `문자`로 출력한다.
2. `<냥 N개>!` <br>
변수 N의 값을 `십진수 형태`로 출력한다.
3. `<냥 N개>!!` <br>
변수 N의 값을 `ASCII 코드`로 해석하여 해당 `문자`로 출력한다.
4. `냐!` <br>
현재 스택을 보여주는 내장함수이다.
5. `냐!!` <br>
현재 변수 테이블을 보여주는 내장함수이다.

예시:
```
...!    # 3을 출력 
냥!      # 변수1의 값을 숫자형으로 출력
냥냥!!   # 변수2의 값을 아스키 코드로 출력
```
