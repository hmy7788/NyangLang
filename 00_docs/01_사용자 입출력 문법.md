# NyangLang 사용자 입출력 문법
본 문서는 NyangLang의 사용자 입출력 문법을 정의한다.
1. 입력
2. 출력

---

## 1. 입력(input)
형식:
```
<냥 N개>?
```
의미:
- 사용자로부터 정수 입력을 받아 변수 N에 저장한다.

예시:
```
냥?     # 변수1 ← 입력
냥냥?   # 변수2 ← 입력
```

## 2. 출력(output)
형식:
```
1. <냥 N개>!
2. <냥 N개>!!
```
의미:
1. `<냥 N개>!` <br>
변수 N의 값을 십진수 형태로 출력한다.
2. `<냥 N개>!!` <br>
변수 N의 값을 ASCII 코드로 해석하여 해당 문자로 출력한다.

예시:
```
냥!      # 변수1의 값을 숫자형으로 출력
냥냥!!   # 변수2의 값을 아스키 코드로 출력
```
